# ApexPay - Banking App 

A modern Next.js web application for managing bank accounts, transactions, and users.
Built with React and TypeScript, this project shows a clean full stack structure as a digital banking platform prototype.



# Features
	•	User Authentication – Secure sign-up, login, and session handling
	•	Bank Accounts – Create, view, and manage multiple accounts
	•	Transactions – Deposit, withdraw, and transfer between accounts
	•	Transaction History – Full tracking of all past transactions
	•	Responsive UI – Optimized for both desktop and mobile devices
